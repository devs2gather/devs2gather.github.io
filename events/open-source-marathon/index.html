Who should apply?

The marathon welcomes participants of varying skill levels and backgrounds. Enthusiastic students and learners passionate about open-source, wants to work on real world projects and make a career in tech.

Students looking to expand their network and join open source community or projects.


What you will gain?

  1. Learn valuable coding and collaboration skills.
  2. Be part of a community that values diversity, creativity, and innovation.
  3. Build a strong portfolio showcasing open source contributions.
  4. Foster networking and personal growth while making a meaningful impact.

Participation

Participating in this marathon leads to personal growth, professional opportunities, and community building. But it all begins with meaningful contributions to open-source software.


Contributors

Here's what you need to know to participate and complete this marathon

  • Register anytime between this program timeline, 19 August - 15 October
  • Pull request can be made in any GitHub hosted open source project
  • Project maintainers must accept your pull/merge requests for them to count toward your total
  • Have 3 pull/merge requests accepted between program timeline to complete this event

FAQ

Frequently asked questions

[+]

We have exams in mid of the program can we participate after exams?

Yes, you can register anytime between the program timeline and start contributing.

[+]

What is this workshop about?

This is not a workshop but a 8 weeks long event where you have to contribute to open-source projects. We will teach you how to contribute, and you will have to choose projects and contribute to them.

[+]

What are the training topic and timings?

We will provide taining on topics like Git, GitHub, contribution workflows, etc. We will publish the training schedule soon.

[+]

I don't have much programming knowledge, can I participate in this event?

Open-source contribution is not only about code contributions, you still have lots of ways to contribute to open-source projects. You can contribute to documentation, translation, design, etc. So, yes you can participate in this event.
